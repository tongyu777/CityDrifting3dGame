# CityDrifting3dGame

Browser-based 3D drifting game built with Three.js.

Enjoy fun drifting in the city!

![Drift Demo](images/202605.gif)

## Gameplay

![Screenshot](images/Screenshot.png)

## Controls

- **WASD** → Vehicle control
- **Spacebar** → Drift / handbrake
- **C** → Switch between first-person and third-person camera

## Features

- Physics tuned to mimic real drift cars
- Counter-steering behavior
- First-person & third-person camera modes
- Custom 3D city environment
- Drift handling and vehicle dynamics
